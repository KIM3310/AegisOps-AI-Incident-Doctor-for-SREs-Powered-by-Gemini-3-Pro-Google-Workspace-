# Live Session Pack

`/api/live-session-pack` is the generic realtime operator surface for AegisOps.

Use it when you need to explain:

- what enters a live incident bridge
- which human roles stay in the loop
- how voice, screenshots, logs, and structured handoff stay aligned
- which review routes must be checked before trusting the live path

Core routes:

- `/api/healthz`
- `/api/runtime/scorecard`
- `/api/live-session-pack`
- `/api/review-pack`
- `/api/schema/report`

The pack is intentionally generic. It is suitable for realtime incident response demos without locking the product narrative to a single vendor or challenge.
